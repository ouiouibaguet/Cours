Les deux types de balises

il esxiste deux types de balises :

  1. Les balises visuelles encadrent du code, par exemple : <p> et <p>
  2. Les balises dites orphelines n'encadrent aucun code ; elles se ferment elles-mêmes. Par exemple : <br/> abréviation de break row; et <hr/> abréviation de horizontal row;
